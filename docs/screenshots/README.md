# Lumina — Marketplace Screenshots

Professional screenshots for the Salla theme marketplace listing.
Use these (or re-shoot from your own live store via `salla theme preview`)
when filling the theme metadata in **Salla Partners → Themes**.

| File | Use |
|------|-----|
| `01-homepage.png` | Hero / main listing thumbnail (desktop + mobile) |
| `02-product-page.png` | Product page — sticky buy box & conversion features |
| `03-mini-cart.png` | AJAX mini-cart drawer (mobile) |
| `04-category-page.png` | Category listing with filters & 4-col grid |
| `05-dark-mode.png` | Dark mode showcase |

> Tip: Salla usually asks for a cover image + several gallery shots.
> Recommended export size: 1600×1000 (or 1280×800) PNG/JPG.
